# Durgesh — SAVEE Training

Author: Durgesh Singh (durgesh@Durgeshs-MacBook-Air.local)  
Dataset: SAVEE  
Model: SVM with GridSearchCV  
Accuracy: ~80.21%  
Emotions: angry, disgust, fear, happy, neutral, sad, surprise (7 classes)

## Files
- `train_savee.ipynb` — training notebook
- `train.py` — training script with GridSearchCV
- `extract_features.py` — feature extraction utility
- `predict.py` — predict from audio file
- `model.pkl` — saved model bundle (model + scaler + label encoder)
- `requirements.txt` — dependencies
